# Installtion 
## 1. Im Codespace
1. Installiere FFMPEG 
```bash
sudo apt update
sudo apt install ffmpeg
```
2. Installiere Python Packages
```bash
pip install -r requirements.txt
```
3. Installiere die Python Erweitung für VS Code

## 2. Lokal auf Windows
1. Installiere FFMPEG
   - Lade FFMPEG von der offiziellen Webseite herunter: https://ffmpeg.org/download.html
   - Entpacke die heruntergeladene Datei und füge den `bin`-Ordner zum Systempfad hinzu.
2. Installiere Python Packages
```bash
pip install -r requirements.txt
```

## 3. Lokal auf MacOS
1. Installiere FFMPEG mit Homebrew:
   ```bash
   brew install ffmpeg
   ```
2. Installiere Python Packages
```bash
pip install -r requirements.txt
```

# Nutzung
1. Starte das Hauptskript:
```bash
python main.py
```
